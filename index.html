Cómo editar:
  1. En tu editor (Codespace o GitHub web) abre este archivo index.html.
  2. Busca el bloque que empieza con %%{init: y edita el texto del flowchart (nodos y flechas usan la sintaxis de Mermaid).
  3. Guardar / Commit → GitHub Pages reconstruirá la página y verás el cambio en la URL pública.

Ejemplo rápido: para agregar "Topología Algebraica" como nodo conectado a "Análisis Complejo" añade una línea debajo de los nodos: TopAlg["Topología Algebraica"]:::leaf y una flecha Complex --> TopAlg.
